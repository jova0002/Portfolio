TOPIC

This first module provides a basic introduction to the most commonly used tools in the multimedia designer's toolbox. These tools, such as basic professional concepts in digital user interfaces, digital content production, digital communication and responsive web design, form the foundation for the rest of your programme at MMD.

GOAL

We learn how to set up web pages according to html and CSS and gain first-hand skills in creating graphics and image processing, setting up text and images in Figma

PROCESS

Core Elements

throughout this topic we had to follow a given structure for the layout of a website which would challenge us in different ways. One of these challenges was creating out first gallery layout using grids. The design process which we had to do was explore colour pallets and find 5 colours, as well as two font families which would be displayed on our site

Mobile and Desktop

We got introduced to the importance of making a website both mobile and desktop friendly. we made two different sites, one for mobile and one for desktop, which was a challenge to learn as we had to understand the basics of how to use @media queries in relation to the max/min width of a site

CODING

Grids to Gallery

Creating the gallery display, I had to learn the different uses of display:grid and the way it alters the layout on the site. This step took a while to come past as i had to learn how a grid layouts need very specific direction within the code to understand how it needs to be displayed and that there is a numerical set up within grids that tells the code exactly its position.

JavaScript

This was our first introduction to JavaScripts where we were shown how to make a “burger menu” for the header of a site. This is important especially within mobile sites to be able to create a smooth navigation for its users. I had to learn how to add an “EventListener” and understand the use of JavaScript which is the foundation behind any interactive item on any site you see.

RESULTS

Learnings & Solutions

Throughout this process I gained my first knowledge on how to approach developing a functioning website and learning throughout our time of self study how to tackle problems along the way. The final result of our site displays the new skills I learned throughout this topic such as creating functioning links, buttons, layouts and grids which I take on with me to my next projects
