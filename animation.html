TOPIC

In this topic you will be introduced to the javascript programming language, which can be used to code dynamic websites. We analyse game design and look at how to engage users through the theory and conventions of UI elements, character elements, background design and composition techniques

GOAL

Your game must have a concept and idea that you come up with yourself and you must design all elements and code the game from scratch. This course emphasises that you start working professionally with the design of graphic elements

PROCESS

Moodboard

I started by creating a moodboard based off the idea I had inspired by Harry Potter. I wanted there to be a sense of mystery and warmth and to capture the feeling of a fantasy world with a mix of dark academia. by creating a moodboard through my process it supports me in having a closer understanding of the creative direction I want to take.

Styletile

This brought me on to make a rough version of a styletile where I had chosen a colour pallete that consisted of darker tones on the warmer side of the colour wheel. I also mocked up a logo in adobe illustrator that gave me an idea of the font style I wanted to capture and my games aesthetic

Adobe Illustrator / UI Elements

Using Adobe illustrator for the first time through this project really challenged my skills as an designer. this was going to be the entire look of our mini game so I had to learn how to use this source to my advantage and spent hours learning all the different tools that are available to me in order to capture the specific vision I had.

These were the final results of my UI elements within my game. These were made by using the pen tool, gradients and the paintbrush tool.

CODING

JavaScript

This topic had a main focus on us delving into the possibilities of JavaScript. I learned how to manipulate elements, handle user inputs, and use logic to create dynamic and responsive gameplay. It was the hardest coding I had done throughout my semester and broadened my mind on how much truly goes into creating movements and interactive elements on a website

Voiceover

using JavaScript, I learned how to incorporate my own voice recordings into my mini game and code them so they are instructed to play when a user clicks upon an element or a time frame runs out, e.g. game over

RESULTS

Learnings & Solutions

Creating my Harry Potter inspired mini-game using JavaScript and Adobe Illustrator taught me how to combine coding and design to create an engaging and interactive experience. I learned to use JavaScript for game logic, user interactions, and dynamic elements, while Adobe Illustrator allowed me to design custom visuals and assets that matched the theme of the game. This project enhanced my technical skills, encouraged creativity, and gave me valuable insight into how code and design work together to create cohesive digital projects.
